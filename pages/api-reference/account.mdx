# Register User

Register a new user account.

## Endpoint

**POST** `/auth/register`

## Request Parameters
The request should include the following parameters in JSON format:

| Parameter    | Type   | Required | Description                 |
| ------------ | ------ | -------- | --------------------------- |
| `username`   | string | Yes      | The user's username         |
| `first_name` | string | Yes      | The user's first name       |
| `last_name`  | string | Yes      | The user's last name        |
| `email`      | string | Yes      | The user's email address    |

## Response
If the request is successful, the API will respond with an HTTP status code of 201 and a JSON object containing the following:

| Parameter    | Type   | Description                                                 |
| ------------ | ------ | ----------------------------------------------------------- |
| `statusCode` | number | HTTP status code returned by the API (e.g. 201 for success) |
| `message`    | string | A message indicating the outcome of the API call            |
| `payload`    | object | A JSON object containing any data returned by the API       |

### Example Request

```json
POST /auth/register
{
    "username": "johndoe",
    "first_name": "John",
    "last_name": "Doe",
    "email": "johndoe@example.com",
}
```

### Example Response

```json
HTTP/2 201 CREATED
{
    "statusCode": 201,
    "message": "Created",
    "payload": {
        created: true,
    }
}
```
