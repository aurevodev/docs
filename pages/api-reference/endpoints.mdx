## User

### `GET` Get User - `/user`

Retrieves information related to the currently authenticated user.

<details>
  <summary>Show More</summary>
  <Tabs items={['cURL', 'Response', 'Response Codes']}>
  <Tab>
  ```sh copy
    curl --request GET \
         --url https://api.aurevo.us/user \
         --header 'Authorization: Bearer <TOKEN>'
  ```
  </Tab>
<Tab>
  ```json copy
{
    "statusCode":200,
    "message":"Success",
    "payload":{
        "platform_uuid":"string",
        "image_hash":"string",
        "first_name":"string",
        "last_name":"string",
        "username":"string",
        "email":"string",
        "country":"string",
        "balance":"number",
        "metadata":"string[]",
        "verified":"boolean",
        "limit":"number",
        "role":"string",
        "created_at":"string",
        "updated_at":"string"
    }
}
```
</Tab>
<Tab>
| Code  | Description                             |
| ----- | --------------------------------------- |
| `200` | Successful response.                    |
| `401` | Authorization header is missing.        |
| `403` | Invalid or expired authorization token. |
</Tab>
</Tabs>
</details>

### `POST` Create Database - `/databases`

Create database for the current authenticated user.

<details>
  <summary>Show More</summary>
  <Tabs items={['cURL', 'Response', 'Response Codes']}>
  <Tab>
  ```sh copy
     curl --request POST \
              --url https://api.aurevo.us/databases \
              --header 'Authorization: Bearer <TOKEN>' \
              --header 'Content-Type: application/json' \
                   --data '{
                       "name": "string",
     	                 "region": "string",
     	                 "type": "string"
                     }'
  ```
  </Tab>
<Tab>
  ```json copy
{
	"message": "Created",
	"payload": {
		"platform_uuid": "string",
		"container_id": "string",
		"name": "string",
		"region": "string",
		"type": "string",
		"connection_url": "string",
		"zone": "string",
		"created_at": "string"
	},
	"statusCode": 201
}
```
</Tab>
<Tab>
| Code  | Description                             |
| ----- | --------------------------------------- |
| `201` | Resource Created.                       |
| `400` | Bad request.                            |
| `401` | Authorization header is missing.        |
| `403` | Invalid or expired authorization token. |

</Tab>
</Tabs>
</details>

### `DELETE` Delete Database - `/databases/`

Delete database using the `id` of the current authenticated user.

<details>
  <summary>Show More</summary>
  <Tabs items={['cURL', 'Response', 'Response Codes']}>
  <Tab>
  ```sh copy
     curl --request DELETE \
              --url https://api.aurevo.us/databases \
              --header 'Authorization: Bearer <TOKEN>' \
              --header 'Content-Type: application/json' \
                   --data '{
                      "id": "string"
                  }'
  ```
  </Tab>
<Tab>
  ```json copy
{
	"message": "Success",
	"payload": {
		"deleted": true
	},
	"statusCode": 200
}
```
</Tab>
<Tab>
| Code  | Description                             |
| ----- | --------------------------------------- |
| `200` | Resource Deleted.                       |
| `401` | Authorization header is missing.        |
| `403` | Invalid or expired authorization token. |
| `404` | Not found.                              |

</Tab>
</Tabs>
</details>
