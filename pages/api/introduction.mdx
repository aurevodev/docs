# API Reference

The Aurevo API is organized around REST. Our API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

```
# BASE URL:

https://aurevo.us/api
```
